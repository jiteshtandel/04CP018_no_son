Widget for bbPress which expands the output of Recent Topics to show Forum and Topic Excerpt.

== Description ==

This plugin requires bbPress.  Tested on version 2.5.3

This plugin can be used to replace the (bbPress) Recent Topics widget.

It adds the additional functionality to display the Forum that a Recent Topic belongs in, and the ability to display the Topic Excerpt.
Additional functionality will be added to this plugin, which makes formatting the output of the widget more flexible.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin using "add new" from the plugins menu item on the Wordpress control panel. Search for bbPress Advanced Recent Topics and install.
OR
1. Download the zip file and extract
2. Upload `bbpress-advanced-recent-topics` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu.
4. Add it to your site via the 'Widgets' menu
